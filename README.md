# UART Transmitter and Receiver using Verilog HDL

## Overview

This project implements UART Transmitter and Receiver using Verilog HDL.

Features:

- FSM based design
- RTL implementation
- UART serial communication
- Testbench verification
- Synthesizable design

## UART Frame

START BIT

↓

8 DATA BITS

↓

STOP BIT
